## 📚 Sobre este Repositorio

¡Hola! Este es mi espacio dedicado a recopilar todo el aprendizaje, ejercicios y proyectos desarrollados durante mi formación académica en **Ingeniería en Informática**. 

Aquí encontrarás desde los conceptos básicos de sintaxis hasta implementaciones más complejas de lógica y estructuras de datos aplicadas.

## 🛠️ Contenido del Proyecto

*   **Fundamentos:** Variables, bucles, y condicionales.
*   **POO:** Clases, herencia, polimorfismo e interfaces.
*   **Algoritmos:** Resolución de problemas lógicos y matemáticos.
*   **Configuraciones:** Archivos de entorno y gestión de versiones.

## 🚀 Tecnologías y Herramientas

*   **Lenguaje:** Java (OpenJDK)
*   **IDE:** IntelliJ IDEA 
*   **Control de Versiones:** Git & GitHub
